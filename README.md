# AI Voice Tutor

## Overview
This project is a real-time AI Voice Tutor that supports:
1. Free-flow voice-based AI chatbot
2. Interactive roleplay-based learning

## Features
- Voice input using Speech-to-Text
- AI-powered responses
- Text-to-Speech output
- Scenario-based roleplay mode

## Tech Stack
- Python / JavaScript
- Speech-to-Text
- Large Language Model
- Text-to-Speech

## How It Works
1. User speaks via microphone
2. Speech is converted to text
3. AI generates contextual response
4. Response is converted back to speech

## Use Cases
- Interview practice
- Language learning
- AI tutoring
